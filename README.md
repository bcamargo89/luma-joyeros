# LUMA Joyeros - Joyería de Lujo

Sitio web moderno, elegante y responsivo para joyería de lujo.

## ✨ Características

- 🎨 **Diseño Elegante**: Paleta dorada, blanca y gris oscuro
- 📱 **Totalmente Responsivo**: Optimizado para todos los dispositivos
- 🛒 **E-commerce Completo**: Carrito, wishlist, filtros avanzados
- 🌍 **Acceso Global**: Desplegado en GitHub Pages
- 🎭 **Slider Hero**: Múltiples imágenes con cambio automático
- 💎 **Personalización Visual**: Herramientas interactivas

## 🚀 Demo en Vivo

**URL del sitio**: https://bcamargo89.github.io/luma-joyeros

## 🛠️ Tecnologías

- HTML5, CSS3, JavaScript ES6+
- CSS Grid y Flexbox
- Google Fonts y Font Awesome
- LocalStorage API
- Intersection Observer API

## 📁 Estructura del Proyecto

```
Luma/
├── index.html          # Página principal
├── styles.css          # Estilos y animaciones
├── script.js           # Funcionalidades interactivas
├── .github/workflows/  # GitHub Actions para deploy
└── README.md           # Documentación
```

## 🚀 Cómo Usar

1. **Clona el repositorio**:
   ```bash
   git clone https://github.com/bcamargo89/luma-joyeros.git
   cd luma-joyeros
   ```

2. **Abre en el navegador**:
   - **Local**: Abre `index.html` en tu navegador
   - **En línea**: Visita https://bcamargo89.github.io/luma-joyeros

## 🎯 Funcionalidades Principales

- **Slider Hero Automático**: Cambio de imágenes cada 5 segundos
- **Carrito de Compras**: Gestión completa de productos
- **Wishlist**: Lista de deseos persistente
- **Filtros Avanzados**: Por categoría, precio y tipo
- **Chat en Vivo**: Asistencia al cliente
- **Conversión de Monedas**: BOB, USD, EUR
- **Diseño Responsivo**: Optimizado para móviles
